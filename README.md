# Symbolica

Symbolica (Символика) -- простая игра в жанре 2D рогалика. Название игры происходит от слова символ, потому что игра состоит из простых ANSCII символов. В этой игре не предусматривается сюжет, играйте столько, сколько хочется, сохраняйтесь и бейте свои же рекорды на различных платформах.

---

# Описание

В игре на данный момент есть только однопользовательский режим бесконечной игры. Сама игра представляет собой бродилку-выживалку в 2D мире циферок и буковок.

---

# Возможности игры

1. В игре есть возможность ходить в любом направлении! Кроме хождения сквозь стены.

# Дополнительно

Данный проект был реализован в рамках курсов подготовки "C++ Coding Garden" от компании 7bits, далее продолжившийся как пет-проект.

Суть проекта заключалась в понимании основополагающих элементов разработки видеоигр, а именно ECS, различных менеджеров объектов, проектировании уровней с использованием математических функций, а также расширение навыков программирования на C++, настройка кросс-платформенных проектов на CMake и использование сторонних библиотек.
